=== Description ===
An additional component that can know the number of views of a post, in addition to a column dedicated to views within the posts table.

This plugin is used to find out the number of post views.

In addition to a custom column inside the post display table.

Features include:
* Displays the number of post views in a custom column.
* Help bloggers know how many views a post has.

= Installing manually: =

1. Unzip all files to the `wp-content/plugins/view-post` directory
2. Log into WordPress admin and activate the 'view-post' plugin through the 'Plugins' menu
3. Go to a post and see the number of views on it.
